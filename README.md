# CompareHgvs

This project contains two tools that are useful for benchmarking Nirvana HGVS against a truth data set:

 - CompareHgvs
 - ExtractGenesFromNirvanaJson

## CompareHgvs

```
dotnet CompareHgvs.dll COSMIC_3.2.5.json.gz JSON/COSMIC.json.gz NcbiGeneIds.txt
```

The arguments are as follows:

 1. Nirvana JSON path
 2. HGVS Truth JSON path
 3. NCBI Gene ID keep list
